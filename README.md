# Задача №5. Частота слов

Скрипт принимает путь к файлу с текстом и выводит 10 наиболее употреблямых слов.

Для использования необходимо запустить программу в консоли и передать аргументом путь до файла.

Пример использования:
```sh
cd 5_lang_frequency
$ python lang_frequency.py -h
usage: lang_frequency.py [-h] -f FILEPATH

Принимает текстовый файл в качестве аргумента и выводит 10 самых популярных
слов в нем.

optional arguments:
  -h, --help            show this help message and exit
  -f FILEPATH, --filepath FILEPATH
                        Путь до файла.
```
